# RFID-Card-Reader

RFID Card Reader built using  Nodemcu, RFID-Reader RC522, Arduino IDE and XMPP .

## Connections

### Nodemcu with RFID RC522

| Nodemcu ESP8266 | RFID RC522 |
| --------------- | ---------- |
| D2              | SDA/SS     |
| D5              | SCK        |
| D7              | MOSI       |
| D6              | MISO       |
| GND             | GND        |
| D1              | RST        |
| 3V/3V3          | 3.3V       |
